User Stories:

  • make GET request to /block/[blockheight] to get block data at a particular blockheight
  • make POST request to /block with block body data to add a new block to blockchain and return newly added block

Example usage:

[GET] /block/0

Example output:

{"hash":"49cce61ec3e6ae664514d5fa5722d86069cf981318fc303750ce66032d0acff3","height":0,"body":"First block in the chain - Genesis block","time":"1530311457","previousBlockHash":""}

Example Usage:

[POST] [BLOCKBODY="Testing block with test string data"] /block

Example output:

{"hash":"ffaffeb2330a12397acc069791323783ef1a1c8aab17ccf2d6788cdab0360b90","height":1,"body":"Testing block with test string data","time":"1531764891","previousBlockHash":"49cce61ec3e6ae664514d5fa5722d86069cf981318fc303750ce66032d0acff3"}
